# Car Rental System

## Project Overview

The Car Rental System is a full stack web application developed using Java, Spring Boot, Hibernate, and MySQL.

This application allows users to view cars, book rental cars, manage customer details, and track bookings online.

---

## Features

- User Registration and Login
- View Available Cars
- Car Booking System
- Admin Dashboard
- Customer Management
- Booking History
